Project : The Last Standing

Client : KDS-03 (XII IPS 3) SMA 1 Kudus | Year : 2014

Topic : This video is intended as a class memento.

User & Audience : Students of class XII IPS 3

Requirement :

  • Laptop
  • Adobe Photoshop
  • Vegas Pro

Progress :

  1. Collecting media such as photos and videos to be put together in the form of a video work as a memento of one class.
  2. Brainstorming is done when editing begins, which is divided into segments.
  3. When editing is complete, some friends are selected to view the video if there are any corrections.
  4. Furthermore, the corrected video will be converted into a small size and distributed only to classmates.
